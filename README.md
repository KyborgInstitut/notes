# Kyborg Institut – Konfigurationsanleitungen

Willkommen im offiziellen Repository des **Kyborg Instituts** für wiederverwendbare technische Kurzanleitungen.

Hier sammeln wir praxisnahe, markdown-basierte Anleitungen für komplexe oder seltene Konfigurationen – z. B. Netzwerk-Hardware, Server-Setups, Recovery-Prozesse und Tools.

Ziel: Wenn man nach Monaten oder Jahren wieder dasselbe machen muss, findet man hier eine zuverlässige, getestete Anleitung.

---

*(Alle Anleitungen basieren auf realen Durchläufen und werden laufend gepflegt.)*

*Kyborg Institut – Technik verständlich und reproduzierbar.*
